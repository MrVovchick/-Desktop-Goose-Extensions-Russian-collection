# 🪿 Desktop Goose Extensions — Русская сборка
Desktop Goose Extensions — это полностью русифицированная сборка для более комфортной и расширенной игры в Desktop Goose.
В неё добавлены новые мемы, текстовые сообщения и полезные модификации, делающие гуся ещё веселее и интерактивнее.

# ✨ Что нового в сборке
🌍 Русификация
- Полный перевод игры на русский язык
- Переведены мемы и текстовые сообщения
- Переведены изображения внутри игры

# 🧩 Добавленные моды и возможности
Моды расширяют поведение гуся и добавляют новые механики:

- 🪶 Атмосфера

Осенние листочки, которые гусь разбрасывает по экрану (мод идёт вместе с игрой)

- 🔊 Управление

Кнопка F — гусь громко гагает

- 🍞 Взаимодействие

Хлебные крошки (Right Shift)
Можно кормить гуся. Используется кастомная текстура хлеба.

- 🧸 Игрушки

Добавлены предметы для игры с гусем:
палка,
мячик,
лазерная указка.

- 🛏 Комфорт

Лежанка для гуся — позволяет ему отдыхать и не мешать вам во время работы.

- 🖱 Управление гусем

Гуся можно перетаскивать курсором мыши

- 🎮 Discord

Отображение в Discord Rich Presence, что вы играете в Desktop Goose

- 🎩 Косметика

Гусь носит красивый чёрный цилиндр
